# Third-Party Skills Import Ledger

First imported on 2026-07-05. Expanded for OPC founder workflows on 2026-07-06.

This repository includes selected third-party open-source skills. Each imported skill is copied as a complete skill directory, with nonessential evaluation cache files excluded. Upstream license files are preserved under `licenses/`.

Some upstream skills used frontmatter fields outside the current Codex validator schema. Those fields were preserved under `metadata` during import.
