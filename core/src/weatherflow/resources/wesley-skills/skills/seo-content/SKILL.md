# SEO Content

SEO-optimized blog posts, keyword strategy, meta descriptions, and content calendars that drive organic traffic.

## Purpose

Create content that ranks in search and converts readers. This skill combines keyword targeting with compelling writing — not keyword stuffing.

## Workflow

### Step 1: Gather Context
- Target keyword or topic
- Business/product context (what you sell, who you serve)
- Content goal (traffic, leads, authority, sales)
- Existing content (are you competing with yourself?)

### Step 2: Keyword Analysis
- Primary keyword + search intent (informational, commercial, transactional)
- 5-8 secondary/related keywords
- Suggested heading structure using keyword clusters
- Estimated difficulty and volume (qualitative: low/medium/high)

### Step 3: Content Outline
- Title (include primary keyword, under 60 characters)
- Meta description (150-160 characters, includes keyword, has a CTA)
- H2/H3 structure covering the topic comprehensively
- Word count recommendation based on competing content
- Internal linking opportunities

### Step 4: Write the Post
- Hook opening (no "In today's world..." openers)
- Subheadings every 200-300 words
- Short paragraphs (2-4 sentences)
- Natural keyword usage — never forced
- Actionable takeaways the reader can use immediately
- Strong conclusion with CTA

### Step 5: SEO Checklist
- [ ] Primary keyword in title, H1, first 100 words, meta description
- [ ] Secondary keywords in H2s and body naturally
- [ ] Meta description under 160 characters with CTA
- [ ] Internal links to 2-3 related posts
- [ ] External links to 1-2 authoritative sources
- [ ] Alt text for images (if applicable)
- [ ] URL slug is short and keyword-rich

## Output Format

```markdown
## SEO Content: [Topic]

### Target Keywords
- Primary: [keyword] (intent: [type])
- Secondary: [keyword list]

### Meta
- **Title tag:** [under 60 chars]
- **Meta description:** [150-160 chars]
- **URL slug:** /[slug]

### Article
[Full article in markdown]

### SEO Checklist
[Completed checklist]
```

## Constraints
- Write for humans first, search engines second
- Never keyword stuff — if it sounds unnatural, rewrite
- Don't promise specific rankings or traffic numbers
- Always match search intent — don't write a guide when people want a product
